CampusBuddy  

Angular, TypeScript, PostgreSQL, REST API

Worked on a group project of 4 people that lets students to interact with other students, on and off campus. The website is made using Angular Js. It has functionalities such as registering a user, creating a profile, signing in, finding other students and more. The website uses PostgreSQL as its database. The website supports a chat feature that let students message each other. It uses a socket server to connect to database to perform CRUD operations on REST APIs.

  • Designed UML, Class, Use Case and Sequence Diagrams
  • Worked on Work Breakdown Structure
  • Scheduled the project on MS Project
  • Worked on designing and creating sql database tables
  • Worked on Developing Posting section of website
  • CRUD operations on RESTful APIs
  • Worked on Profile Creation part
  • Worked on User Manual and Documentation
