Sumo2Unity  

Unity3D, C#, Python, Sumo

Worked on a integrating transportation software called Sumo to Unity to produce 3D simulation of traffic to experience on VR. The software uses TraCI to receive data to Unity Realtime. TraCI uses a TCP based client/server architecture to provide access to sumo.

  • Established a realtime data transfer connection using TraCI between Sumo and Unity.
  • Contributed to the creation of immersive virtual reality experiences for various traffic scenarios.
  • Implemented various algorithms(Culling Methods) and techniques to improve the overall performance of the Unity project, resulting in a 35% improvement in frame rate.
  • Uses GitHub to maintain code and version control.
  • Worked on User Manual and Documentation

CampusBuddy  

Angular, TypeScript, PostgreSQL, REST API

It has functionalities such as registering a user, creating a profile, signing in, finding other students and more. The website uses PostgreSQL as its database. The website supports a chat feature that let students message each other. It uses a socket server to connect to database to perform CRUD operations on REST APIs.
